File Formatting

Formatting the text file for group submission is very simple and can be done in a few steps.

Creating text file on Windows

1. Right click desktop background or in an open folder window.

2. Click 'New' near the bottom of the popup, then click 'Text Document'.

3. Type in name of the file, or press enter for the default name, the name does not matter.

Creating text file on MacOs

1. Open the textEdit app from the application window/folder or search for it by pressing command + space.

2. Click 'new document'.

3. In the upper left click 'file' and save the text file somewhere onto the computer.

How to format the groups:

1. Open the text file you just made. Insert the names of the students to the text file.

2. Seperate the names of the students into groups so each group is on its own line.

3. Add a comma between the names of the students in the groups and put a period after the last student.
